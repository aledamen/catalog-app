# Fase-Beta Catalogo

Catalogo de productos hecho con Next.js 14, Tailwind CSS y Zustand, listo para deployar en Vercel.

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
npm run start
```

## Deploy en Vercel

1. Subi este proyecto a un repositorio Git.
2. Importalo en Vercel.
3. Vercel detecta Next.js automaticamente.
4. Deploy sin variables de entorno obligatorias.
